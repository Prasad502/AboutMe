#1. Attendance Management using Face-Recognition (using Python , tkinter and OpenCV)

Attendance Management using Face-Recognition

The Attendance management system is a replacement project for the traditional ways of taking attendance in schools , colleges ,etc. This system uses biometrics .i.e Face-Recognition to automatically mark the attendance. The application is made to be used by only the faculty and have overall control on all the features such as updating and deleting the students from the database. The system compares the run time image of the student with an pre recorded image in the system and marks the attendance on the basis of the result of the coomparison. Languages used are python for back-end , Tkinter for front-end , OpenCV for Face-Recognition and MySQL for database.

#2. RECHARGE SYSTEM (using Python-EEl and HTML,CSS,JS)

MOBILE RECHARGE SYSTEM

The Recharge management system is a basic system with various modules integrated to achieve a functionality of adding money to a virtual wallet and then use the virtual money to recharge on the number the user registered with and after recharge check the balance remaining in the wallet. The language used is Python for back-end ; HTML ,CSS ,JS for front-end ; SQLite3 for database and Python-EEL for the data communication.
